최근 AI로 이것저것 자동화를 테스트해보면서

자동화 완료 후 알람을 받기위해 어떤 방법이 있을까 고민하다가

이러한 알람은 텔레그램을 많이들 추천하길래 사용해봤습니다.

## 1. 텔레그램 봇이란

텔레그램 봇은 텔레그램 안에서 동작하는 자동화 계정입니다.

메시지를 보내고 받을 수 있고, 이미지/파일 전송, 커맨드 응답 등

일반 사용자가 하는 대부분의 행동을 API로 할 수 있습니다.

별도 서버 없이도 단순한 알림 용도로 쓸 수 있고

서버를 붙이면 사용자 입력에 응답하는 대화형 봇도 만들 수 있습니다.

## 2. 봇 생성

다른 서비스들처럼 개발자 콘솔에서 등록하고 인증받고 하는 과정이 아니라

텔레그램 앱 안에서 대화로 봇을 만들 수 있어서 신기했습니다.

### 순서

1. 텔레그램에서 `@BotFather` 검색 - 검색하면 공식마크가 있어야하고 로봇같은 할아버지 집사 아이콘입니다.
2. `/newbot` 명령어 입력
3. 봇 이름 설정 (표시 이름)
4. 봇 username 설정 (`_bot`으로 끝나야 함)
5. **Bot Token** 발급 완료

### Chat ID 확인

봇한테 메시지를 보내려면 대상의 Chat ID가 필요합니다.

1. 방금 만든 봇에게 아무 메시지나 전송
2. 브라우저에서 아래 URL 접속

```
https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates
```

3. 응답 JSON에서 `result[0].message.chat.id` 값이 Chat ID

이 토큰과 Chat ID만 있으면 메시지를 보낼 준비는 끝입니다.

## 3. 메시지 보내기

별도 라이브러리 설치 없이 fetch 하나면 됩니다.

```js
const TELEGRAM_BOT_TOKEN = 'your_bot_token';
const TELEGRAM_CHAT_ID = 'your_chat_id';

async function sendMessage(message) {
    const response = await fetch(
        `https://api.telegram.org/bot${TELEGRAM_BOT_TOKEN}/sendMessage`,
        {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({
                chat_id: TELEGRAM_CHAT_ID,
                text: message,
            }),
        }
    );
    return response.json();
}

await sendMessage('알림이 도착했습니다!');
```

이게 끝입니다. 토큰 넣고, Chat ID 넣고, 텍스트 넣으면 메시지가 갑니다.

## 4. 메시지 포맷팅

기본 텍스트만 보내도 되지만 `parse_mode`를 쓰면 서식을 넣을 수 있습니다.

### HTML 모드

```js
await fetch(`https://api.telegram.org/bot${TELEGRAM_BOT_TOKEN}/sendMessage`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        chat_id: TELEGRAM_CHAT_ID,
        text: '<b>완료 !!!</b>\n<a href="https://naver.com">바로가기</a>',
        parse_mode: 'HTML',
    }),
});
```

### 지원 태그

| 태그 | 결과 |
|---|---|
| `<b>굵게</b>` | **굵게** |
| `<i>기울임</i>` | *기울임* |
| `<code>코드</code>` | `코드` |
| `<pre>코드블록</pre>` | 코드 블록 |
| `<a href="url">링크</a>` | 링크 |
| `<s>취소선</s>` | ~~취소선~~ |

### Markdown 모드

`parse_mode: 'MarkdownV2'`를 쓸 수도 있는데

특수문자를 전부 이스케이프 해야 해서 HTML 모드가 더 편합니다.

### 인라인 키보드 버튼

메시지 아래에 클릭 가능한 버튼을 붙일 수 있습니다.

```js
body: JSON.stringify({
    chat_id: TELEGRAM_CHAT_ID,
    text: '완료!',
    reply_markup: {
        inline_keyboard: [
            [
                { text: '바로가기', url: 'https://naver.com' },
                { text: '무시', callback_data: 'ignore' },
            ],
        ],
    },
})
```

`url`을 넣으면 링크 버튼, `callback_data`를 넣으면 봇이 응답을 받을 수 있는 버튼이 됩니다.

## 5. 이미지/파일 전송

텍스트 외에 이미지나 파일도 보낼 수 있습니다.

### 이미지 보내기 (sendPhoto)

```js
// URL로 이미지 보내기
await fetch(`https://api.telegram.org/bot${TELEGRAM_BOT_TOKEN}/sendPhoto`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        chat_id: TELEGRAM_CHAT_ID,
        photo: 'https://example.com/image.png',
        caption: '스크린샷입니다',
    }),
});
```

로컬 파일을 보내려면 `multipart/form-data`로 보내야 합니다.

```js
const fs = require('fs');
const FormData = require('form-data');

const form = new FormData();
form.append('chat_id', TELEGRAM_CHAT_ID);
form.append('photo', fs.createReadStream('./screenshot.png'));
form.append('caption', '로컬 스크린샷');

await fetch(`https://api.telegram.org/bot${TELEGRAM_BOT_TOKEN}/sendPhoto`, {
    method: 'POST',
    body: form,
});
```

### 파일 보내기 (sendDocument)

```js
const form = new FormData();
form.append('chat_id', TELEGRAM_CHAT_ID);
form.append('document', fs.createReadStream('./report.pdf'));
form.append('caption', '리포트 파일');

await fetch(`https://api.telegram.org/bot${TELEGRAM_BOT_TOKEN}/sendDocument`, {
    method: 'POST',
    body: form,
});
```

sendPhoto와 동일한 방식이고 `photo` 대신 `document`를 쓰면 됩니다.

## 6. 봇 커맨드 만들기

지금까지는 봇이 일방적으로 메시지를 보내는 용도였는데

사용자가 봇에게 `/start`, `/help` 같은 명령어를 보내면 응답하는 기능도 만들 수 있습니다.

### 커맨드 등록

먼저 BotFather에서 커맨드 목록을 등록합니다.

1. BotFather에게 `/setcommands` 입력
2. 대상 봇 선택
3. 커맨드 목록 입력

```
start - 봇 시작
help - 도움말
status - 현재 상태 확인
```

이렇게 등록하면 사용자가 `/`를 입력했을 때 자동완성 목록에 나타납니다.

### Polling으로 메시지 받기

봇이 메시지를 받으려면 주기적으로 getUpdates를 호출하는 **polling** 방식을 씁니다.

```js
let offset = 0;

async function pollUpdates() {
    const response = await fetch(
        `https://api.telegram.org/bot${TELEGRAM_BOT_TOKEN}/getUpdates?offset=${offset}&timeout=30`
    );
    const data = await response.json();

    for (const update of data.result) {
        offset = update.update_id + 1;
        const text = update.message?.text;

        if (text === '/start') {
            await sendMessage('안녕하세요! 봇이 시작되었습니다.');
        } else if (text === '/help') {
            await sendMessage('사용 가능한 명령어:\n/start - 봇 시작\n/help - 도움말\n/status - 상태 확인');
        } else if (text === '/status') {
            await sendMessage('정상 동작 중입니다.');
        }
    }
}

// 3초마다 polling
setInterval(pollUpdates, 3000);
```

`timeout=30`은 long polling으로 새 메시지가 올 때까지 최대 30초 대기하는 옵션입니다.

`offset`을 업데이트해야 같은 메시지를 중복 처리하지 않습니다.

### Webhook 방식

polling 대신 텔레그램이 내 서버로 직접 요청을 보내는 webhook 방식도 있습니다.

서버가 있다면 webhook이 더 효율적이지만

간단한 봇이라면 polling으로 충분합니다.

## 7. 마무리

텔레그램 봇은 진입 장벽이 많이 낮았습니다.

텔레그램 앱 UI에서

BotFather로 봇 만들고, 토큰 받고, Chat ID까지만 확인하면

fetch로 메시지를 보낼수 있고

포맷팅이나 이미지 전송, 커맨드 응답까지 별도 라이브러리 없이 REST API만으로 가능합니다.

자동화 스크립트의 알림 용도로는 충분하고

커맨드를 붙이면 간단한 대화형 봇까지 만들 수 있어서

사이드 프로젝트에서 알림이 필요할 때 한번 써보면 좋을 것 같습니다.
